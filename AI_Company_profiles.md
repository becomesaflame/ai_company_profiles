# **AI Company Profiles**

This document serves as a reference guide to key companies operating within the Artificial Intelligence ecosystem, categorized by their primary function or market segment. Each entry includes a brief description of the company's core services and offerings in the AI world.

# **Hyperscalers**

Hyperscalers are large cloud providers that offer a vast range of computing, storage, networking, and machine learning services on a massive scale. They form the backbone of modern cloud-based AI infrastructure.

| Company | Blurb |
| :---- | :---- |
| Amazon Web Services (AWS) | Offers a comprehensive suite of AI and ML services, including Amazon SageMaker for building, training, and deploying models at scale, Amazon Bedrock for accessing foundation models from multiple providers, and pre-trained AI services for computer vision, natural language processing, and forecasting. |
| Microsoft Azure | Provides end-to-end cloud computing services with a strong focus on AI, including Azure Machine Learning for MLOps, Azure AI Services for vision, speech, and language, and Azure OpenAI Service providing enterprise access to GPT-4, DALL-E, and other OpenAI models. |
| Google Cloud Platform (GCP) | Known for its strong foundation in AI research, GCP offers Vertex AI as a unified platform for MLOps and generative AI, Gemini models for multimodal AI applications, deep learning infrastructure including TPUs, and pre-built AI services for translation, speech, and vision. |

# **Neoclouds**

Neoclouds are newer, often more specialized cloud providers that focus on niche markets or offer alternative infrastructure solutions, frequently emphasizing superior performance for specific workloads like deep learning.

| Company | Blurb |
| :---- | :---- |
| CoreWeave | A specialized cloud provider focusing on GPU-accelerated computing infrastructure using NVIDIA GPUs, optimized for AI/ML workloads including large language model training and inference, VFX rendering, and compute-intensive tasks with high-performance networking. |
| Lambda Labs | Offers cloud computing services and on-premise hardware specifically designed for deep learning and AI research, providing access to high-end NVIDIA GPUs with pre-configured environments for popular ML frameworks, targeting researchers, startups, and enterprises. |
| Vultr | A global cloud infrastructure provider offering a full range of cloud services including compute, storage, and networking, with a focus on ease of use, affordability, and global presence, including dedicated cloud GPU instances for AI development. |
| Crusoe | Focuses on sustainable, energy-efficient computing by leveraging stranded energy sources (such as flared natural gas) to power modular data centers, offering high-performance GPU infrastructure primarily for AI model training and large language model development. |
| RunPod | Developer-focused GPU cloud platform providing on-demand access to 30+ GPU types through dedicated instances and serverless endpoints, featuring FlashBoot technology for industry-leading cold start times and dual infrastructure options from community-sourced to enterprise-grade data centers. |
| Hot Aisle | An AMD-exclusive cloud provider offering on-demand bare metal and virtualized MI300X/MI355x GPU instances with no-contract provisioning, enterprise compliance (SOC2, HIPAA), and 100% renewable energy operations optimized for AI/ML workloads and LLM training. |

# **GPU Marketplaces**

GPU marketplaces are platforms that allow users to rent or lease GPU resources from a decentralized network of providers, making high-powered computing accessible and flexible for burst workloads or cost-effective AI training.

| Company | Blurb |
| :---- | :---- |
| Fluidstack | A decentralized computing platform that allows users to rent high-performance GPUs globally for AI training, rendering, and other intensive computations from a network of providers. |
| Vast AI | Offers a vast marketplace for renting and selling decentralized cloud GPU computing power, catering to individuals and organizations looking for flexible and cost-effective AI infrastructure. |
| Akash Network | A decentralized, open-source cloud computing marketplace that allows users to buy and sell cloud resources, including GPU compute, utilizing blockchain technology for transparency and efficiency. |
| Aranya | A platform for decentralized or peer-to-peer sharing of specialized AI hardware, connecting developers with underutilized, high-end compute clusters globally for cost-effective training and inference. |
| Shadeform | A GPU cloud marketplace that aggregates access to GPU instances across 20+ cloud providers through a unified API and console, providing real-time availability and pricing data for optimal resource allocation. |
